== Edit Application

You can edit your aown applications.


===
****

. Click My APPs tab
. Select application to edit
. Click ... dot on the right upper corner
. Click Edit APP button
- You can edit following fields:

* APP name
* Description
* Active or Inactivate
* Organization Id
* Directory
